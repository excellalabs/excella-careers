Overview

The Business Intelligence Developer Lead is responsible for leading teams to enable technical solutions that support self-service analytics environments for our clients.

Responsibilities

BI Tool Selection & Deployment

  • Interacts directly with client stakeholders (business or technical) to understand stakeholder needs and current environment to recommend best-fit BI tools and platforms.
  • Able to lead formal BI Tool selection and evaluation process when required
  • Works with chosen BI Tool vendor to install chosen solution
  • Tailors installation to client stakeholder needs

 

BI Content Delivery

  • Able to perform the following work and/or lead teams of BI Developers to perform this work:
    • Uses SQL and other query tools to understand raw data sources identified.
    • Works with architect and data integration developer to design data structures that support report and dashboard requirements (i.e. has the data elements available, query performance is acceptable etc.)
    • Creates simple to complex reports and dashboards using BI Tools like Tableau, Cognos, BO etc.
    • Creates prototypes based upon stakeholder requirements in order to confirm the report or dashboard design meets the business need.
    • Where applicable, designs and builds OLAP structures (cubes) to support BI deliverables using tools like SSAS, Cognos Cube Designer etc.
    • Uses SQL connecting directly to data source in order to confirm counts shown in BI Tool.
    • Ensures the final BI solution follows best practices, is automated and is easily transitioned into the client's environment for ongoing maintenance.
    • Creates and delivers end-user training and documentation on BI deliverables.

 

Qualifications

Qualifications (Technical)

 

  • 10+ years developing reports or dashboards using Business Intelligence Tools (e.g. Tableau, Business Objects, Cognos, Qlikview, Microstrategy).
  • 10+ years writing simple to complex SQL queries.
  • Expert understanding of relational, multidimensional (OLAP) and Big Data structures (Hadoop/Spark) in order to source effectively from these for reports and dashboards.
  • Expert in BI tool administration functions including:
  • Publishing of content to shared environment
  • User access and tool security
  • Data mapping/catalog changes (semantic layer)
  • Archive and purge activities
  • Performance tuning assessment and implementation
  • Other tool maintenance, including installation of patches
  • Experience working hands on in multiple BI platforms
  • Experience with performance tuning SQL, including the use of explains plans, partitions, hints etc.

 

Qualifications (Non-Technical)

 

  • Excellent communication skills to be able to interact directly with non-technical client stakeholders and act in a business-to-technical translation role.
  • Proficient leading and mentoring small teams of technical professionals.
  • Experience deploying BI platforms.
  • Experience working within the Agile Scrum Framework.
  • Self-motivated and self-managing.
  • Proficient in participating in design and code reviews.
  • Proficient in identifying and communicating scope changes.
  • Proficient in creating reasonable and accurate time estimates for assigned tasks.
